# Исследования рынка общепита в Москве для принятия решения об открытии нового заведения

Исследование рынка на основе открытых данных о заведениях общественного питания, рекомендации для открытия новой кофейни. 

*Сфера деятельности*: стартапы, бизнес

*Навыки и инструменты*: python, pandas, folium, json, plotly, seaborn, matplotlib, визуализация данных
